# telemapp
---

### How to use

 - Run the `run_telemetry.sh` script from the root directory of the project to compile and upload the arduino firmware and start the docker container. To do this:

     1. Clone the repository from GitHub to an accessible location
     2. Navigate to the repository and go to the `Container` directory using `cd Container`
     3. Run the shell script using this syntax: `./run_telemetry.sh`

 - Navigate to `http://localhost:5000` in a browser to access the UI.
