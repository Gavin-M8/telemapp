How to Use:

  • Start: Begin receiving and plotting telemetry data. Data will appear on the chart in real time.

  • Stop: Pause data reception and plotting. The reader thread stops temporarily.

  • New CSV Log: Start logging data to a new CSV file with a timestamped filename. Also resets performance statistics.

  • Stop Logging: Stop writing to the current CSV file. Plotting continues uninterrupted.

  • Reset Stats: Reset all performance metrics (max g-forces, velocity, 0-60 timer, event counts) without stopping logging.

  • Delete All Logs: Permanently remove all CSV files from the logs folder. Use with caution!

  • Performance Metrics: The dashboard shows real-time derived telemetry including max g-forces, estimated velocity, 0-60 mph time (when accelerating hard), and event detection for corners and braking.

  • G-G Diagram: Shows your vehicle's acceleration envelope - the outer boundary represents maximum performance in different directions.
