How to Use:

  • Start: Begin receiving and plotting telemetry data. Data will appear on the chart in real time.
  • Stop: Pause data reception and plotting. The reader thread stops temporarily.
  • New CSV Log: Start logging data to a new CSV file with a timestamped filename. Data continues to plot while logging.
  • Stop Logging: Stop writing to the current CSV file. Plotting continues uninterrupted.
  • Delete All Logs: Permanently remove all CSV files from the logs folder. Use with caution!
  • The chart displays the Ax, Ay, Az accelerometer axes in real time.
  • The Status indicator shows whether the reader is RUNNING or STOPPED.
  • The Current CSV indicator shows which file is currently being logged (or none).
  • Time on the X-axis is shown as HH:MM:SS.mmm for easy readability.
