How to Use

  • Start: Begin telemetry reading from the device or dummy data.
  • Stop: Pause telemetry reading without losing previous data.
  • New CSV Log: Stop current logging, start a new CSV file with timestamped name, and resume reading.
  • Delete All Logs: Permanently remove all CSV files from the logs folder.
  • The chart shows Ax, Ay, Az accelerometer axes in real time.
  • Status and Current CSV indicate the reader state and log file.
